  2. 下载 离线安装包
    在搜索引擎上搜 “chrome 42 离线安装包“ 即可

  3. 用7zip 解压

  4. 继续解压

  2. 找到chrome.exe 的位置
    我的是在C:\Program Files (x86)\Google\Chrome\Application。将上一步得到的两个文件夹拷贝过去， chrome.exe 覆盖
